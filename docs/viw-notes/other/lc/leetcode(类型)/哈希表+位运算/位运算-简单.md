# 位运算





## 介绍

**运算符：**

- `&` : 或，都为1 结果为1 。 ex : `11&01=01`

- `|`  且，一个为1 就为1. ex: `11|01=11`

- `~`  非，1为0，0为1  ex: `~1=0  ~0=1  `

- `^` 异或，相同为0，不相同为1， ex: `11^01= 10`

- `<<` 左移。 ex: ` 1<<2 1左移2位 相当于 100`  

- `>>` 右移。 ex `1>>2 1右移2位 相当于 1 还右移就为0了`

**位操作：**

- (3&1)== 1  == >  (3%2)==1
- (2&1)== 0 ==  > (2%2) == 0
