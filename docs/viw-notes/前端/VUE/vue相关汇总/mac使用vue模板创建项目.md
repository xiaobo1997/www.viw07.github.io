

# 1.创建模板项目

**前提**
- node
- npm
- vue

1.node&npm

`brew info node`
`brew info npm`

![](https://xiaoboblog-bucket.oss-cn-hangzhou.aliyuncs.com/blog/20220413113104.png)


2.换源
`sudo npm install -g cnpm --registry=https://registry.npm.taobao.org`

3.vue-cli

`sudo cnpm install -g @vue/cli`

3.查看版本
`vue -V`

4.创建project

`vue create vue-demo`

5.运行
`cd vue-demo`
`npm run serve`
