# surfing快捷键


> 写在前面
chrome plugins 快捷键可以让使用vim操作浏览器页面 常用的并不多，记录一下




## 复制当前页面地址
```json   
yy
```

## 当前滚动页面 向下

```json   
j
```

## 当前滚动页面 向上


```json   
k
```

## 进入可视模式 来操作页面

```json   
zv
```

### 上一行下一行

```json   
k: 上一行
j: 下一行
```

### 单词

```json   
w:前进一个单词
b:后退一个单词
```


## 浏览器搜索

```json   
og：使用google搜索
ow：使用bing搜索
ob: 使用baidu搜索
```
