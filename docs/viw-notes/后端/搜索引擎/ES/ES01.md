### 添加文档

发送post请求 ，在某一个索引下面  发起json 的请求体

如 url:  http://localhost:9200/shopping/_doc

请求体:

```json
{
    "id":1,
    "name":"电脑",
    "price":"100"
}
```
> es会生成随机的id

指定我们自己的id 添加doc

如 url:  http://localhost:9200/shopping/_doc/1

![](https://xiaoboblog-bucket.oss-cn-hangzhou.aliyuncs.com/blog/20220105222430.png)

**注意 content type 是`application/json`**


> `http://localhost:9200/shopping/_create/1` 也可以 type 也是 _doc

### 修改

#### 全量修改

发送put请求，是幂等请求， 如url: `http://localhost:9200/shopping/_doc/1` 和添加时一样，发送请求成功以后可以看到，版本号有变化，并且result 是update


#### 部分修改

发送post请求，如 url: `http://localhost:9200/shopping/_update/1`

```json
{
    "doc":{
        "price":"110",
    }
}

```

### 复杂查询


#### 条件

> 可以发送 `http://localhost:9200/shopping/_search?q=price=110` 这样不可取


发送get请求，通过设置请求content-type 为 json

url: `http://localhost:9200/shopping/_search`

```json

{
    "query":{
        "match":{
            "name":"电脑"
        }
    }
}

```

**全量查询**
```json

{
    "query":{
        "match_all":{
            
        }
    }
}

```


#### 分页


url: `http://localhost:9200/shopping/_search`

```json

{
    "query":{
        "match_all":{

        }
    },
    "form":0,
    "size":2
}
```

> form是起始索引，从0开始，size是页容  比如上面的 就是  每一页2条数据，从下标0开始查


#### 查询指定字段


url: `http://localhost:9200/shopping/_search`

```json

{
    "query":{
        "match_all":{

        }
    },
    "form":0,
    "size":2,
    "_source":["name"]
}
```


#### 排序



url: `http://localhost:9200/shopping/_search`

```json

{
    "query":{
        "match_all":{

        }
    },
    "form":0,
    "size":2,
    "_source":["name"],
    "sort":{
        "price":{
            "order": "desc"
        }
    }
}
```

### 多条件查询&范围查询

**组合多条件查询:**

method:GET
url: http://localhost:9200/shopping/_search
content-type:json

```json

{
    "query":{
        "bool":{                        条件
            "must":[                    必须
                {                       
                    "match":{           匹配
                        "name":"手机"    属性
                    }
                },
                {                       
                    "match":{           匹配
                        "price":"120"   属性
                    }
                }
            ]
        }
    }
}

```

> must 有必须匹配  相当于 and ，如果是 or  可以 把 must 换成 should



结果:
```json

{
    "took": 34,                         耗时
    "timed_out": false,                 是否超时
    "_shards": {
        "total": 1,
        "successful": 1,
        "skipped": 0,
        "failed": 0
    },
    "hits": {
        "total": {
            "value": 1,
            "relation": "eq"
        },
        "max_score": 1.605183,
        "hits": [                       命中
            {
                "_index": "shopping",
                "_type": "_doc",
                "_id": "3",
                "_score": 1.605183,
                "_source": {
                    "id": 3,
                    "name": "手机",
                    "price": "120"
                }
            }
        ]
    }
}

```


**范围查询:**

```json

{
    "query":{
        "bool":{                        条件
            "should":[                  or
                {                       
                    "match":{           匹配
                        "name":"手机"    属性
                    }
                },
                {                       
                    "match":{           匹配
                        "price":"120"   属性
                    }
                }
            ],
            "filter":{
                "range":{
                    "price":{
                        "gt":100
                    }
                }
            }
        }
    }
}


```


### 全文检索&完全匹配&亮度查询


**全文检索**

可以参考viw-simple-lab项目中的代码 和 viw-mall的项目代码中es那一块
和solr的分词


**完全匹配**

```json

{
    "query":{
        "match_phrase":{
            "name":"笔记本"
        }
    }
}

```

> match_phrase  必须全文匹配，不分词的去检索

**高亮**


```json

{
    "query":{
        "match_phrase":{
            "name":"笔记本"
        }
    },
    "highlight":{
        "fields":{
            "name":"电脑"
        }
    }
}

```



### 聚合查询

```json
    "aggs":{                        //聚合

            }
        }
    },
    "size":0                        //原始数据展示条数
}

```
